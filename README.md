# Curavie
An AI-powered Electronic Health Record (EHR) system.

## Tech Stack
- Backend: Python (Flask/Django)
- Frontend: HTML, CSS, JS
- Database: MySQL (planned)
- AI: NLP + LLM integration (future)

## Project Structure
- `/frontend` → HTML, CSS, JS
- `/backend` → Python (Flask/Django)
- `/docs` → project documentation
